# BangoPoints Loyalty Platform - Minimalist Demo Prompt

## ðŸŽ¯ Project Goal
Create a lightweight, fully functional browser-only loyalty rewards platform demo with 6 user roles, zero backend dependencies, and minimal npm packages.

## ðŸ“¦ Minimal Tech Stack
- **Frontend**: React 18 + Vite (fast, modern bundler)
- **Styling**: CSS-in-JS using styled-components OR pure CSS modules (NO Tailwind needed)
- **Routing**: React Router DOM v6 (essential for multi-page navigation)
- **State**: React Context API (NO Zustand - use built-in React)
- **Icons**: Unicode symbols OR inline SVG (NO icon libraries)
- **HTTP**: Fetch API with custom mock interceptor (NO Axios)
- **Charts**: HTML5 Canvas OR CSS-only visualizations (NO Recharts)

**Total Dependencies: 2 (react-router-dom + vite)**

### 2. Zero-Dependency Mock API (api.js)
```javascript
// Lightweight fetch wrapper with intelligent mocking
const MOCK_MODE = true;
const API_DELAY = 300; // Simulate network latency

const mockHandlers = {
  '/auth/login': (body) => ({ success: true, user: DEMO_DATA.users[body.role], token: 'demo-' + Date.now() }),
  '/shoppers/points': () => ({ success: true, data: { points: 2450, tier: 'silver', multiplier: 1.5 } }),
  '/shoppers/receipts': () => ({ success: true, data: DEMO_DATA.receipts }),
  '/shoppers/rewards': () => ({ success: true, data: DEMO_DATA.rewards }),
  '/admin/users': () => ({ success: true, data: Object.values(DEMO_DATA.users) }),
  '/admin/brands': () => ({ success: true, data: DEMO_DATA.brands }),
  '/admin/stores': () => ({ success: true, data: DEMO_DATA.stores }),
  '/brand-manager/analytics': () => ({ success: true, data: DEMO_DATA.analytics }),
  '/ppg/clock': (body) => ({ success: true, message: body.action === 'in' ? 'Clocked in' : 'Clocked out' }),
  '/beo/receipts/pending': () => ({ success: true, data: DEMO_DATA.receipts.filter(r => r.status === 'pending') })
};

export const api = {
  get: async (url) => {
    if (MOCK_MODE) {
      await new Promise(resolve => setTimeout(resolve, API_DELAY));
      const handler = mockHandlers[url];
      return handler ? handler() : { success: false, error: 'Not found' };
    }
    const res = await fetch('/api' + url);
    return res.json();
  },
  
  post: async (url, body) => {
    if (MOCK_MODE) {
      await new Promise(resolve => setTimeout(resolve, API_DELAY));
      const handler = mockHandlers[url];
      return handler ? handler(body) : { success: false, error: 'Not found' };
    }
    const res = await fetch('/api' + url, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify(body) });
    return res.json();
  }
};
```

### 5. Context-Based State Management (AuthContext.jsx)
```javascript
// Simple, powerful state management with React Context
import { createContext, useContext, useState } from 'react';

const AuthContext = createContext();

export const AuthProvider = ({ children }) => {
  const [user, setUser] = useState(null);
  const [loading, setLoading] = useState(false);
  
  return (
    <AuthContext.Provider value={{ user, setUser, loading, setLoading }}>
      {children}
    </AuthContext.Provider>
  );
};

export const useAuth = () => useContext(AuthContext);
```
